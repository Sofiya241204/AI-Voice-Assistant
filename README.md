# AI Voice Assistant

A smart voice-controlled assistant built using Python that can perform basic tasks through voice commands.

## Features

* Voice recognition using microphone
* Text-to-speech responses
* Open Google
* Open YouTube
* Play songs on YouTube
* Tell the current time
* Search information from Wikipedia

## Technologies Used

* Python 3.10
* SpeechRecognition
* pyttsx3
* PyWhatKit
* Wikipedia
* Webbrowser

## Project Structure

```text
AI-Voice-Assistant/
│
├── assistant.py

```

## Installation

Install the required libraries:

```bash
pip install speechrecognition
pip install pyttsx3
pip install pywhatkit
pip install wikipedia
pip install pyaudio
```

## Run the Project

```bash
python assistant.py
```
